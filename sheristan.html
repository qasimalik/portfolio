What is Sheristan?

Sheristan is a full-stack mobile platform for reading, writing, and sharing Urdu and Hindi poetry. It supports user authentication, content creation, bookmarking, social interactions, and offline access. The application is built end-to-end including mobile frontend, backend APIs, database design, and production deployment.

Application Architecture

  • React Native mobile application for user interface.
  • Spring Boot backend exposing REST APIs.
  • MongoDB for persistence and indexing.
  • Dockerized services running on AWS EC2.
  • CI/CD pipeline for automated builds and deployments.
