Project Overview

Cruddur is a cloud bootcamp project focused on deploying a micro-blogging platform using AWS infrastructure, containerization, and cloud networking fundamentals. The project emphasizes hands-on exposure to cloud provisioning, deployment workflows, and operational practices.

Technical Scope

  • AWS EC2 provisioning and environment setup.
  • Docker container builds and runtime execution.
  • Cloud networking fundamentals and firewall configuration.
  • Backend and frontend service integration.
  • Environment variables and runtime configuration management.
  • Basic deployment automation.
